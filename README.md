It is a verilog code for same size parameterized signed matrix multiplication block with a self verifying sytem verilog testbench          
Mulblock  performs multiplication of two elements       
Pe block contains  mul block and a adder.It performs multiplication of n elements of row and n elements of column and add them to get a single element of output matrix
Macblock it contains  which is used to calculate a row or 4 elements of output matrix parallelly.
